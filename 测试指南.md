## 🐛 调试技巧

```bash

adb logcat -d > test_log.txt

# 查看崩溃
adb logcat | grep FATAL
```

### 检查权限
```bash
# 检查无障碍服务
adb shell settings get secure enabled_accessibility_services

# 检查悬浮窗权限
adb shell dumpsys window | grep "draw-over-ops"
```

## ⚠️ 重要提示

### 必须配置
1. **无障碍服务** - 在设置中启用
2. **悬浮窗权限** - 授予系统权限

### 兼容性
- ✅ Android 5.0+ (API 21+)
- ✅ 支持真机和模拟器
- ✅ 免Root

### 设备建议
- **真机测试** - 功能最完整(推荐)
- **模拟器** - 方便快速测试
- **内存** - 至少2GB RAM

## 💡 常见问题

**Q: 没有Android Studio怎么办?**
A: 下载官方版本,免费且自带所有必要工具

**Q: 没有真机能测试吗?**
A: 可以!使用内置模拟器,Tools → Device Manager

**Q: Gradle同步很慢?**
A: 首次需要下载依赖,耐心等待或使用国内镜像

**Q: 点击不生效?**
A: 检查无障碍服务是否启用,这是必须的

**Q: 想打包成APK?**
A: `./gradlew assembleRelease` (需要签名配置)
